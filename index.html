Download

There are two ways to get the latest version of la4j: download zipped archive from Google Code Downloads or add new dependency to your Maven's pom.xml. 

<dependency>
  <groupId>org.la4j</groupId>
  <artifactId>la4j</artifactId>
  <version>0.2.0</version>
</dependency>


Overview

The la4j - is single threaded, open source and 100% Java library witch oriented to solve main problems of Linear Algebra. There are key features of la4j, listed bellow:

  • minimalism (60kb jar)
  • uniform interpretation of vectors and matrices
  • sparse (CSR, CSC) and dense (1d, 2d arrays) matrices and vectors support
  • linear systems solving (Gaussian, Jacobi, Zeidel, Square Root, Sweep and other)
  • matrices decomposition (Eigenvalues, SVD, QR, LU, Cholesky and other)
  • inverted matrices foundation
  • serialization for matrices and vectors
  • MatrixMarket/CSV output format

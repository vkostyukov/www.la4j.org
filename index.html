Download

There are two ways to get the latest version of la4j: download zipped archive from Google Code Downloads or add new dependency to your Maven's pom.xml. 

<dependency>
  <groupId>org.la4j</groupId>
  <artifactId>la4j</artifactId>
  <version>0.2.0</version>
</dependency>


Overview

The la4j - is single threaded, open source and pure Java library witch oriented to solve main problems of Linear Algebra. There are key features of la4j, listed bellow:

  • no dependencies and tiny size (~60kb jar)
  • minimalistic and full object-oriented API
  • sparse (CSR, CSC) and dense (1d/2d arrays) matrices
  • linear systems solving (Gaussian, Jacobi, Zeidel, Square Root, Sweep and other)
  • matrices decomposition (Eigenvalues, SVD, QR, LU, Cholesky and other)
  • MatrixMarket/CSV input/output formats support for matrices and vectors

Samples

Factories

Factory basic2d = new Basic2DFactory();
Factory csr = new CSRFactory();

double array[][] = new double[][] { 
    {1.0, 0.0, 0.0}, 
    {0.0, 5.0, 0.0}, 
    {0.0, 0.0, 9.0} 
};

Matrix a = basic2d.createMatrix(array);
Matrix b = csr.createMatrix(array);

// convert dense 2D-array to sparse CSR matrix
Matrix c = a.copy(csr);

// convert sparse CSR to dense 2D-array matrix
Matrix d = b.copy(basic2d);
Tip! Don't use factories everywhere. Try public constructors!

Rows and Columns

Matrix a = new CSRMatrix(array);

int rows = a.rows();
int columns = a.columns();

// changes number of columns/rows
a.setColumns(columns + 1); 
a.setRows(rows + 1);

// resizes the matrix
a.resize(rows + 1, columns + 1);

// swaps columns/rows
a.swapColumns(0, columns - 1);
a.swapRows(0, rows - 1);

// row - is a sparse compressed vector
Vector row = a.getRow(0); 

// column - is a sparse compressed vector
Vector column = a.getColumn(0);

Matrix inversion 

Matrix a = new Basic2DMatrix(array);

// b - is a dense 2D-array matrix
Matrix b = a.inverse(new GaussianInvertor());

// c - is a sparse CSC matrix
Matrix c = a.inverse(
    new GaussianInvertor(), 
    new CSCFactory()
);

Input/Output

Matrix a = new Basic2DMatrix(array);

// MatrixMarket format privider
ObjectOutput mmos = new MMOutputStream(
    new FileOutputStream("file.mm")
);
mmos.writeObject(a);
mmos.close();

ObjectInput mmis = new MMInputStream(
    new FileInputStream("file.mm")
);

// b - is a dense 2D-array matrix
Matrix b = (Matrix) mmis.readObject();
mis.close();

Basic operations

Matrix a = new CSCMatrix(array);
Matrix b = new Basic1DMatrix(array);

// c - is a sparse CSC matrix
Matrix c = a.multiply(b);

// d - is dense 1d-array matrix
Matrix d = a.multiply(b, new Basic1DFactory()); 

// e = c + d - a * 100
Matrix e = c.add(d).subtract(a).multiply(100);

Specific operations

Matrix a = new CSCMatrix(array);
Matrix b = new Basic2DMatrix(array);

// calculates the trace of matrix
double trace = a.trace();

// calculates the product of matrix
double product = a.product();

// c - is a sparse CSC matrix
Matrix c = a.transpose();

// d - is a dense 1D-array Matrix
Matrix d = b.transpose(new Basic1DFactory());

// e - is a triangle sparse CSC matrix
Matrix e = a.triangularize();

// f - is a triangle dense 2D-array matrix
Matrix f = a.triangularize(new Bascic2DFactory());

Solving linear systems

Matrix a = new CSRMatrix(array);
Vector b = new BasicVector(array[0]);

LinearSystem system = new LinearSystem(a, b);

// x - is a sparse compressed vector
Vector x = system.solve(new GaussianSolver()); 

// y - is a dense basic vector
Vector y = system.solve(
    new SquareRootSolver(), 
    new Basic1DFactory()
);

Matrix decomposition

Matrix a = new Basic1DMatrix(array);

// qr[0] = Q, qr[1] = R; Q,R - dense 1D-array matrices
Matrix[] qr = a.decompose(new QRDecompositor());

// lu[0] = L, lu[1] = U; L,U - sparse CSR matrices
Matrix[] lu = a.decompose(
    new LUDecompositor(), 
    new CSRFactory()
);
